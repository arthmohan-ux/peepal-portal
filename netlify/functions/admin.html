Deploy Guide

How to apply changes from this panel to the live portal and Google Sheets.

Portal (Netlify) — 4 steps
1

Make your changes here

Edit roles, pipelines, access, people in this admin panel. Nothing is saved yet — it's all in memory.

2

Go to Generate Code

Click "Generate All" — you'll get a code block for each file that needs updating.

3

Paste into your repo

Each block tells you exactly which file and what to replace. For config.js, replace the whole file. For feedback.js and admin-page.js, replace only the constant shown.

4

Push to git

git add . && git commit -m "config: update" && git push — Netlify auto-deploys in ~30 seconds.

Google Apps Script — 3 steps
1

Generate the GAS blocks

Same Generate page — you get blocks for HiringTracker_v1.gs and EmailAutomations.gs.

2

Paste into Apps Script

Sheet → Extensions → Apps Script. Find the constant (e.g. const DEPT_ROLES = {) and replace the whole object. Ctrl+S to save.

3

Run Refresh Config in the sheet

Use 📋 Hiring Tracker → 🔄 Refresh Config to apply dropdown changes to existing rows.
